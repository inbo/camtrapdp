# camtrapdp (development version)

* New accessory function `locations()` to retrieve location information from deployments of a camtrapdp object.
* New accessory functions `deployments()`, `media()` and `observations()` to retrieve deployments, media and observations resources of a camtrapdp object.
* New function `read_camtrapdp()` to read data files into memory.
* New function `version()` to get the version of a camtrapdp object.
* New internal function `example_package()` to get the latest Camtrap DP example 
dataset.
* New internal function `convert()` to convert camtrapdp objects to the latest 
version. This function is currently not used, as the only supported version is 
Camtrap DP 1.0.
