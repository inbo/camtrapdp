# Read the Camtrap DP example dataset

Reads the [Camtrap DP example
dataset](https://camtrap-dp.tdwg.org/example/). This dataset is
maintained and versioned with the Camtrap DP standard.

## Usage

``` r
example_dataset()
```

## Value

Camera Trap Data Package object.

## Examples

``` r
example_dataset()
#> A Camera Trap Data Package "camtrap-dp-example-dataset" with 3 tables:
#> • deployments: 4 rows
#> • media: 423 rows
#> • observations: 549 rows
#> 
#> And 1 additional resource:
#> • individuals
#> Use `unclass()` to print the Data Package as a list.
```
