# camtrapdp: Read and Manipulate Camera Trap Data Packages

Read and manipulate Camera Trap Data Packages ('Camtrap DP'). 'Camtrap
DP' (<https://camtrap-dp.tdwg.org>) is a data exchange format for camera
trap data. With 'camtrapdp' you can read, filter and transform data
(including to Darwin Core) before further analysis in e.g. 'camtraptor'
or 'camtrapR'.
